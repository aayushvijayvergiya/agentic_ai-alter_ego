## Phase 3 ŌĆö Email Notifications

### 3.1 Replace Pushover with Email (`EmailService`)

Add an `EmailService` using Gmail SMTP (via Python `smtplib`):
- Send email to Aayush when `record_unknown_question` tool is called
- Send email when `record_user_details` tool is called (someone left their contact)
- Subject line: `[Alter Ego] Unknown question` / `[Alter Ego] New contact: {name}`
- Keep Pushover as a secondary optional channel (don't remove it)

**New env vars**:
- `NOTIFICATION_EMAIL_TO` ŌĆö Aayush's email address (recipient)
- `GMAIL_USER` ŌĆö Gmail account to send from
- `GMAIL_APP_PASSWORD` ŌĆö Gmail App Password (not account password; requires 2FA + App Password setup in Google account)

**Alternative** (if Gmail SMTP is undesirable): `SENDGRID_API_KEY` with SendGrid free tier.

**Files to create/change**:
- `src/alter_ego/services/email_service.py` (new)
- `src/alter_ego/services/__init__.py` (export)
- `src/alter_ego/config.py` (add email vars)
- `src/alter_ego/services/tool_handler.py` (wire in EmailService alongside NotificationService)
- `tests/test_alter_ego.py` (add `TestEmailService`)

## Phase 4 ŌĆö Portfolio Embedding & UX

### 4.1 Streaming Responses

Replace the single-shot OpenAI call with a streaming approach for better perceived performance:
- Gradio `ChatInterface` supports streaming via Python generator functions
- Tool calls (function calling) are incompatible with streaming ŌĆö implement a two-pass approach:
  1. First call: non-streaming, handle tool calls if present
  2. If no tool calls: stream the final response
- Add `stream=True` to the OpenAI call in the second pass

**Files to change**:
- `src/alter_ego/services/chat_service.py`
- `src/alter_ego/app.py` (update `chat_handler` to be a generator)

### 4.2 Portfolio Website Embedding (iframe)

Configure Gradio to allow cross-origin embedding from the portfolio domain:
- Add `allowed_paths` and `root_path` to `launch()` in `app.py`
- Gradio 5.x supports `allowed_iframe_origins` parameter
- Set env var `PORTFOLIO_DOMAIN` for the allowed origin

Provide an embed snippet in `docs/embedding/README.md`:
```html
<iframe
  src="https://aayush2012-alter-ego.hf.space"
  width="100%"
  height="600"
  frameborder="0"
></iframe>
```

**Files to change**:
- `app.py` (root ŌĆö HF entry point)
- `src/alter_ego/app.py`
- `src/alter_ego/config.py` (add `portfolio_domain`)

### 4.3 Basic Rate Limiting

Prevent OpenAI cost abuse on a public space:
- Track per-session message count using Gradio's `gr.State`
- Soft-block after N messages (configurable via `MAX_MESSAGES_PER_SESSION` env var, default: 20)
- Return a polite message when limit is reached
- Do not implement complex IP-based rate limiting (rely on HF Spaces abuse protection for that)

**Files to change**:
- `src/alter_ego/app.py`
- `src/alter_ego/config.py`
